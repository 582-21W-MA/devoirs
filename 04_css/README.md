# Devoir : CSS

Pour ce devoir, vous devez mettre en forme le document HTML réalisé lors
de l'exercice précédent afin que l'apparence de votre page corresponde à
celle de la [page originale]. Vous pouvez ajouter des `<div>` et des
`<span>` au besoin.
