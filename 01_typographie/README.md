# Devoir : Typographie

Pour ce devoir, vous devez concevoir la typographie pour une nouvelle
version du site Web du groupe de musique microtonale québécois
[Angine de Poitrine]. Votre proposition doit inclure un [style] pour les
titres et un autre pour le corps de texte. Il est interdit d'utiliser
les polices de caractères inclus avec Figma. Référez-vous à ce
[tutoriel] si vous ignorez comment ajouter des fontes.

Veuillez utiliser le texte « [À propos] » pour tester et présenter vos
choix. Votre devoir doit être remis sous forme d'un fichier PDF, et
inclure une courte justification.

[Angine de Poitrine]: https://anginedepoitrine.com
