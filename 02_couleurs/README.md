# Devoir : Couleur

Pour ce devoir, vous devez concevoir la palette de couleur pour une
nouvelle version du site Web de la [boulangerie Fairmount]. Votre
proposition doit inclure au moins une couleur primaire et une couleur
secondaire. Veuillez aussi indiquer les combinaisons permises, et le
taux de contraste pour chacune d'elles. Votre devoir doit être remis
sous forme d'un fichier PDF, et inclure une courte justification.

[boulangerie Fairmount]: https://fairmountbagel.com
