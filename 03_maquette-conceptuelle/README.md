# Devoir : Maquette conceptuelle

Pour ce devoir, votre tâche est de concevoir une maquette conceptuelle
qui représente une solution au problème que vous avez identifié lors de
l'exercice précédent. Concentrez-vous sur les pages du site Web qui sont
concernées par le problème. Pour chacune d'elles, identifiez l'objectif
principal de la page, ainsi qu'une ou deux lois de la Gestalt utilisées
pour structurer son interface.
