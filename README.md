# Devoirs

Vous trouverez dans ce dépôt les énoncés des devoirs. Chaque devoir doit
être remis _avant_ le début du cours suivant.

Les devoirs doivent être téléversés sur GitHub, dans _votre_ propre
dépôt Classroom (et non pas ici). Les devoirs remis ailleurs ne seront
pas comptabilisés. Pour créer votre dépôt, [cliquez ici][Classroom], et
suivez les instructions. Vous serez éventuellement re-dirigé·e vers un
dépôt qui porte votre nom d'utilisateur GitHub.

> [!IMPORTANT]
> Assurez-vous que votre nom d'utilisateur GitHub contienne votre nom de
> famille.

Pour téléverser un devoir, utilisez l'interface web de GitHub:

1. D'abord, assurez-vous que tous les fichiers du devoir se trouvent
   dans un dossier dont le nom correspond au nom du devoir (par exemple,
   « 01_typographie »).
2. Cliquez sur le bouton `Add file` ou `+` (à gauche du bouton vert
   `Code`), puis choisissez l'option `Upload files`.
3. Sélectionnez le dossier où se trouve le devoir, et glissez-le dans le
   rectangle indiqué.
4. Enfin, cliquez sur le bouton `Commit changes`.

Si vous voyez votre dossier dans le dépôt, alors il a été remis
correctement.

On vous rappelle que vous n'avez pas à _compléter_ un devoir pour
recevoir tout vos points. Il suffit de soumettre quelque chose.

[Classroom]: https://classroom.github.com/a/kA-FVKgs
